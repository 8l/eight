eight by diiq

eight is a programming language designed for easy abstraction; it is based on lisp and the closure algebra

Eight is a language designed with the emotional state of the programmer in mind. It appeals to the core of a programmer, by reducing the cost of DSL design to something like adding a new function.

Dependencies

Boehm GC, byacc, flex

Install

Install eight only if you can do so without instructions. Eight is a long way from complete, and documentation is partial.
